# FSPagingKit

## Requirements

iOS 11+
Swift 5
Xcode 14+

## Installation

FSPagingKit is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'FSPagingKit'
```
